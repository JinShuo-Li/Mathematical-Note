数学早在被形式化之前就已经开始。

在学会书写一个 $\varepsilon$--$\delta$ 证明以前，我们已经对连续性具有某种直观感受；在接触群与域以前，我们已经学会计算、整理表达式并辨认对称性；在拓扑学精确定义开性与连通性以前，我们至少已经能够非正式地区分什么彼此相连，什么彼此分离。

这种直觉并不是数学的障碍，而是数学的起点。然而，仅有直觉并不足够。熟悉的图像可能产生误导，看似合理的命题可能在意外的例子中失效，而令人信服的论证也可能隐藏着从未明言的假设。因此，数学的发展始终带有一种持续的张力：我们依靠直觉发现什么可能为真，却依靠定义与证明判定什么确实为真。

这份手稿源于我试图理解从直觉走向严谨的过程。

它最初只是我在学习若干大学数学基础领域时写下的一组笔记。起初目的很实际：记录定义，保存有用的论证，并把在课程与教材中分散出现的结论联系起来。然而，随着笔记不断积累，一个更广阔的结构逐渐显现。分析、代数、几何、拓扑与概率中的思想并不是彼此孤立的构造。它们反复依赖同一些思维习惯：识别正确的对象，选择适当的等价概念，判断哪些性质在变换下保持，并从少量明确假设出发建立一般结论。

本书的目标，就是使这些联系变得可见。

它并不试图成为一部数学百科全书，也不试图取代那些分别深入讨论各个学科的优秀教材。它的范围必然有所选择：有些理论会得到较为详细的展开，另一些则只作为导论或通往后续学习的桥梁出现。目的不是抵达每一个高等结论，而是给出一条贯穿若干核心数学语言的连贯路径，并解释其基本定义与定理为什么采取今天的形式。

因此，各章强调的是结构，而不是彼此孤立的计算。一个定义被引入，并不只是作为需要记忆的术语，而是作为对某个数学问题的回应；一个定理被陈述，也会同时展示使其成立的机制。证明不仅用于确认结论，也用于揭示可以重复使用的方法：逼近、紧致性、完备性、分解、不变性、反证以及取适当极限。例子展示一个概念的适用范围，反例则标出一个看似自然的命题从何处开始失效。

本书的组织体现了一个逐步走向抽象的过程。我们从逻辑与集合的语言开始，因为后续每一种理论都需要精确描述数学对象及其关系。随后进入分析与线性代数，其中极限、连续性、向量空间与线性变换提供了最早的一批重要一般结构。抽象代数把运算从其具体作用对象中分离出来研究；几何与拓扑则以越来越灵活的距离、连续性和不变性概念考察空间。复分析展示了额外的代数结构如何产生出乎意料的强大分析结论；测度论与概率论则把“大小”与积分的语言扩展到初等方法不再充分的情形。

这些学科外观各异，却由一个共同的建筑原则联系在一起。数学通过明确写出假设，借助逻辑论证提取其后果，再追问当假设被削弱、改变或迁移到新的情境时，原有结构中的哪些部分仍然保留，从而不断向前发展。从这个意义上说，数学严谨性不是对直觉的压抑，而是一种纪律：它使直觉能够安全地走出最初产生它的那些例子。

本书面向已经接触过一些大学数学，并希望把各门课程理解为一个更大整体之组成部分的读者。手稿可以从头到尾连续阅读，也可以按需选读。前面的章节提供后文反复调用的语言与技术，而每一章也尽量保持一定程度的独立性。读者应当在定义处停下来，在阅读证明前先尝试自行重构论证，用例子检验假设，并追问删去某个条件后究竟会发生什么。数学很少通过不间断的阅读学会；它更多是通过一次次合上书页、尝试独立复现论证而掌握的。

任何一组笔记都不可能消除所有缺口。有些证明或许可以写得更短，有些解释仍可改进，而内容取舍也不可避免地反映作者自己的学习路径。因此，这份手稿更应被看作一个仍在发展的叙述，而不是最终权威。它的目的并不是让读者毫无疑问地接受每一行文字，而是促使读者更仔细地检查这些文字，与其他来源比较，并独立地把论证继续下去。

在数学的核心，仍然存在我们开头所说的那种张力。我们追求确定性，但问题总会把我们引向尚未被形式化的领域；我们构造定义以使思想精确，而这些定义反过来又揭示直觉本身未曾预见的可能。数学大厦由公理与证明建成，却由好奇心赋予生命。

这本书邀请读者参与这一建造过程：不仅观察已经完成的定理，还要理解地基为什么如此铺设，不同层级如何彼此支撑，以及新的结构仍可能在何处继续生长。
